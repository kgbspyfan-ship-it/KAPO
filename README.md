# KAPO — Сайт адвокатской коллегии «Кузнецов и Партнёры»

Корпоративный сайт адвокатской коллегии в Пскове. Статический многостраничный сайт, задеплоен на GitHub Pages с собственным доменом.

**🌐 Демо:** [https://advocatepskov.ru](https://advocatepskov.ru)

---

## О проекте

Сайт представляет адвокатскую коллегию «Кузнецов и Партнёры» (г. Псков) и команду практикующих адвокатов. Реализованы:

- Главная страница с презентацией коллегии, услугами и формой контакта
- Отдельные страницы адвокатов (например, `kolokoltsev.html`)
- Адаптивная вёрстка
- Кастомная типографика (Playfair Display / Lora / Inter)
- QR-код для быстрой связи

## Стек

- **HTML5** (статическая вёрстка, без сборщиков)
- **CSS3** (CSS-переменные, медиа-запросы, анимации)
- **Google Fonts** (Playfair Display, Lora, Inter)
- **GitHub Pages** (хостинг + кастомный домен через `CNAME`)

## Структура

```
KAPO/
├── index.html          # Главная страница
├── kolokoltsev.html    # Страница адвоката А.С. Колокольцева
```

## Локальный запуск

Сайт полностью статический — достаточно открыть `index.html` в браузере. Для корректной работы относительных путей лучше поднять локальный сервер:

```bash
# Python 3
python -m http.server 8000

# или Node.js
npx serve .
```

После запуска открыть `http://localhost:8000`.

## Деплой

Сайт автоматически публикуется через **GitHub Pages** из ветки `main`. Домен `advocatepskov.ru` подключён через файл `CNAME` в корне репозитория.

Чтобы задеплоить изменения:

```bash
git add .
git commit -m "feat: update site"
git push origin main
```

GitHub Pages обновит сайт в течение ~1 минуты.
